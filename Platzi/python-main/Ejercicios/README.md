# Ejercicios de Python

Bienvenido a la carpeta de **Ejercicios** del curso de Python. Aquí encontrarás una serie de ejercicios diseñados para ayudarte a practicar y consolidar los conceptos que has aprendido a lo largo del curso. Estos ejercicios abarcan desde los fundamentos básicos del lenguaje hasta la aplicación de Programación Orientada a Objetos (POO) y el uso de librerías estándar como `math` y `random`.

## Ejercicio 1: Calculadora Científica Mejorada

**Descripción:**
En este ejercicio, crearás una calculadora científica utilizando POO y la librería `math`. La calculadora permitirá realizar operaciones básicas (suma, resta, multiplicación, división) y funciones matemáticas avanzadas como potencia, raíz cuadrada, logaritmos, y seno/coseno.

**Instrucciones:**
- Crea la clase `Calculadora` con métodos para cada operación.
- Utiliza la librería `math` para las funciones avanzadas.
- Implementa un menú interactivo para que el usuario seleccione la operación que desea realizar.
- Maneja posibles excepciones para evitar errores durante las operaciones.

**Objetivos de Aprendizaje:**
- Aplicar POO en la construcción de una herramienta útil.
- Utilizar la librería `math` para resolver problemas matemáticos.
- Mejorar la interacción con el usuario a través de la consola.

## Ejercicio 2: Sistema de Gestión de Estudiantes

**Descripción:**
Desarrolla un sistema de gestión de estudiantes que permita registrar estudiantes, asignarles calificaciones en diferentes materias, y calcular su promedio general. El sistema también debe categorizar a los estudiantes según su rendimiento.

**Instrucciones:**
- Crea la clase `Estudiante` con métodos para gestionar materias y calificaciones.
- Implementa una clase `SistemaGestion` para administrar la lista de estudiantes y sus registros.
- Calcula promedios y clasifica a los estudiantes según su rendimiento.

**Objetivos de Aprendizaje:**
- Desarrollar habilidades en POO, especialmente en la creación de sistemas de gestión.
- Practicar la manipulación de datos utilizando listas y diccionarios.
- Aplicar lógica para calcular estadísticas y clasificar información.

## Ejercicio 3: Simulación de Lanzamiento de Dados

**Descripción:**
Crea un programa que simule el lanzamiento de dados utilizando POO y la librería `random`. El programa debe permitir realizar múltiples lanzamientos y calcular estadísticas como el promedio de los resultados y la distribución de frecuencias.

**Instrucciones:**
- Crea la clase `Dado` para representar un dado con un número configurable de caras.
- Implementa la clase `SimuladorDados` para manejar los lanzamientos y calcular las estadísticas.
- Usa la librería `random` para generar los resultados de los lanzamientos.

**Objetivos de Aprendizaje:**
- Aplicar conceptos de POO en simulaciones de eventos aleatorios.
- Utilizar la librería `random` para generar valores aleatorios.
- Analizar y calcular estadísticas básicas a partir de datos generados.
